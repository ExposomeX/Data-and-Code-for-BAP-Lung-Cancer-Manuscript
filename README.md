The supplementary codes provide the computational workflows used in this study. Code 1 implements the global benzo[a]pyrene (BaP) reconstruction pipeline used to generate historical and projected exposure fields (Fig. 2 and Fig. S4). Code 2 contains the latency-aware health risk attribution framework and validation procedures used for estimating lung cancer risk under different exposure accumulation strategies (Fig. 3 and Fig. S5–S7). Code 3 constructs the Exposure–Biological–Disease (E-BIO) network to identify shared mechanistic pathways linking BaP and other air pollutants to lung cancer (Fig. 4C and Fig. S10–S11). Code 4 implements the spatial downscaling pipeline for estimating high-resolution global lung cancer burden attributable to air pollution (Fig. 4E).
